Почти во всех современных браузерах есть очень эффективные Developer Tools, только тосты не жарят. Обычно они включатся нажатием F12. Кроме всего прочего, в дебаггер можно попасть используя ключевое слово debugger.

Конечно же, лучше всего тот код, который отлаживать не нужно. Писать юнит тесты, в JavaScript сейчас уже достаточно развита эта область, плюс интеграционные тесты с помощью фреймворков вроде Selenium.

Задание 2

Опишите своими словами, что происходит в браузерах при клике на ссылку yandex.ru (полнота ответа — на ваше усмотрение).

  1. Генерируется событие наведения мыши mouseover
  2. Применяется CSS псевдокласс hover
  3. Генерируется событие нажатия mousedown
  4. Применяется CSS псевдокласс active
  5. Если мышка убрана с элемента, то это расценивается как drag&drop, клик не засчитан
  6. Иначе генерируется событие mouseup и, собственно, click
  7. Если событие по умолчанию для ссылок не прервано (e.preventDefault()), то браузер совершает переход.

Касательно генерации событий: событие создаётся и спускается вниз, последовательно проходя все родительские, элементы до элемента-цели. Эта фаза называется capture фазой. Затем, событие поднимается обратно вверх. Эта фаза называется bubble фазой.

Обработчики событий вызываются в том порядке, в котором были добавлены.
